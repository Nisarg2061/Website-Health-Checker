# Website Health Checker
### A tool developed in go to check website health.
## Steps to setup locally:
  step 1: Install go
  <br>
  step 2: git clone https://github.com/Nisarg2061/Website-Health-Checker.git
  <br>
  step 3: go run . --domain <domain name\>
